# 前端接口文档：获取用户余额

### 1. 接口信息
- **接口地址**: `GET /api/wallet/balance`
- **认证**: 需要登录 (Header 携带 `Authorization: Bearer {token}`)
- **描述**: 仅返回当前登录用户的 ID 和余额，数据轻量，适用于导航栏或个人中心余额实时显示。

### 2. 请求示例
```javascript
fetch('http://localhost:5125/api/wallet/balance', {
  method: 'GET',
  headers: {
    'Authorization': `Bearer ${token}`
  }
})
```

### 3. 成功响应
```json
{
  "userId": 1,
  "balance": 100.00
}
```

### 4. 其它说明
- 如果需要一次性获取用户所有信息（姓名、头像等），请继续使用 `GET /api/auth/me`。
- 本接口响应更快，数据传输量更小。
